# Spline UI Graphic
Unity3d uGUI custom graphic component based on Spline ExtrudeMesh
![](https://github.com/mitay-walle/com.mitay-walle.ui-graphic-spline/blob/main/proof-of-concept.png)
# Proof of Concept
Not ready for production, Script is heavily wip
# Summary
- any Texture
- any Material
- mesh form setup in ExtrudeMesh (road, square, circle)
- uv aligned across spline
# Known issues
- no color support
- flash alpha
- no uv multiplier
- preferred width / height not calculated
